ECON 687 — University of Maryland

The Paper

What Question Does It Ask?

Rankings of journals and universities drive real decisions in hiring, promotion, and funding. They are almost always reported as a single number with no indication of how precisely that rank is estimated.

This paper asks: given that impact factors are themselves estimated with uncertainty, how much uncertainty surrounds the resulting ranks?

The authors construct confidence sets for ranks. Rather than saying a journal is ranked 5th, they report the full range of ranks that cannot be statistically rejected at the 95% level.

Method in Plain Terms

An impact factor is an average computed from a finite number of articles. It carries a standard error. When you rank things by noisy estimates, the ranks themselves are noisy.

The pipeline:

  1. Start with point estimates (impact factors) and their standard errors
  2. Build a covariance matrix capturing estimation uncertainty
  3. Run a bootstrap simulation to generate possible rankings under that uncertainty
  4. Report, for each journal or university, the set of ranks consistent with the data

The method is implemented in csranks, an R package written by the authors.

What Did They Find?

Journals — 30 economics journals ranked by 2011 impact factor:

  • 10 of the 30 cannot be statistically distinguished from the top 5
  • Only 1 journal can be ruled out of the bottom 25

Universities — 100 universities ranked by average publication impact factor:

  • Rankings at the very top and bottom are fairly precise
  • Substantial uncertainty throughout the middle of the distribution

The main takeaway: point rankings overstate what the data can actually tell us.

The four-step logic repeated for both datasets:

Load clean data, filter to sample, run bootstrap, plot results

The Key Statistical Step — csranks

# Step 1: build a covariance matrix from the standard errors
# diag(SE^2) puts variances on the diagonal, zeros off-diagonal
# This assumes estimation errors are independent across journals
Sigma <- diag(jIF$SE^2)

# Step 2: run the bootstrap
CS <- csranks::csranks(
  jIF$IF,           # point estimates
  Sigma = Sigma,    # covariance matrix
  coverage = 0.95,  # 95% confidence level
  stepdown = TRUE,  # iterative procedure, gives tighter sets
  simul    = FALSE, # marginal sets (each journal covered individually)
  R        = 1000,  # bootstrap draws
  seed     = 101    # reproducibility
)

# Step 3: attach bounds to the data frame
jIF$CSL <- CS$L   # lower bound of rank confidence set
jIF$CSU <- CS$U   # upper bound of rank confidence set

From Bootstrap Output to Figure

Once csranks() returns the bounds, the figure follows directly. The bounds become the error bar limits in ggplot.

ggplot(jIF, aes(x = rank, y = reorder(journal, rank))) +
  theme_bw() +
  geom_point(size = 2) +             # dot = point rank estimate
  geom_errorbar(                      # bar = confidence set for rank
    aes(xmin = CSL, xmax = CSU),
    width = 0.4
  ) +
  scale_x_continuous(
    name = "Rank",
    limits = c(1, 30),
    breaks = c(1, 5, 10, 15, 20, 25, 30)
  ) +
  ylab(NULL)

The x-axis is rank, not impact factor. The horizontal bars show the range of ranks the data cannot rule out for each journal at the 95% level.

Marginal vs. Simultaneous

simul = FALSE produces marginal confidence sets. Each journal’s set covers its true rank with 95% probability on its own.

simul = TRUE produces simultaneous sets. All true ranks are covered jointly at 95%. These are wider because the guarantee is stronger.

The paper shows both. Figure 1 uses marginal sets. The appendix shows simultaneous sets for comparison. We replicate both.

What the Figures Tell Us

Both figures show the same pattern the paper describes:

  • Confidence sets are narrow at the very top and very bottom
  • They widen substantially in the middle of the distribution
  • For journals, JEL is clearly ranked first but much of the middle cannot be separated from several adjacent ranks
  • For universities, Chicago and Northwestern sit confidently at the top, but large clusters in the middle are statistically indistinguishable

This is the paper’s central point visualized: point rankings overstate the precision of what we actually know.

Replication Notes

What We Reproduced

Both main figures match the paper. The confidence set bounds were computed using the same csranks package the authors wrote, with identical parameters and the same random seed (101) for reproducibility.

One technical issue we worked through: the current version of csranks requires a covariance matrix rather than a vector of standard errors. The fix:

# This fails with newer csranks:
csranks(jIF$IF, jIF$SE, ...)

# This works:
csranks(jIF$IF, Sigma = diag(jIF$SE^2), ...)

This reflects a deliberate API change by the authors to make the independence assumption explicit rather than assumed silently.
