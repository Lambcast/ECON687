ECON 687 — University of Maryland

The Paper

What Question Does It Ask?

Rankings of journals and universities are used constantly in academic hiring, promotion, and funding decisions. But they are almost always reported as a single number — journal X is ranked 5th — with no indication of how precisely that rank is estimated.

This paper asks: given that the underlying measures (like impact factors) are themselves estimated with uncertainty, how much uncertainty surrounds the resulting ranks?

The authors develop confidence sets for ranks: rather than reporting a single rank, they report the range of ranks that cannot be statistically rejected at the 95% confidence level.

Method in Plain Terms

The core idea is straightforward. An impact factor is an average — it is estimated from a finite number of articles and carries a standard error. When you rank things by noisy estimates, the resulting ranks are also noisy.

The authors’ method:

  • Takes the point estimates (impact factors) and their standard errors as inputs
  • Uses a bootstrap procedure to simulate the distribution of possible rankings
  • Constructs a confidence set for each entity’s rank — the set of ranks that are statistically consistent with the data at the 95% level

The method is implemented in the R package csranks, written by the authors and available on GitHub.

What Did They Find?

Two empirical applications:

Journals — ranking 30 economics journals by 2011 impact factor:

  • 10 of the 30 journals cannot be statistically distinguished from the top 5
  • Only 1 journal can be ruled out of the bottom 25

Universities — ranking 100 universities using publication records from Research Papers in Economics (RePEC):

  • The top and bottom of the ranking are fairly precise
  • Substantial uncertainty throughout the middle

Data Import — What This Code Does

The journal data comes in as a messy Excel file. Each of the 233 sheets is one journal, with rows representing individual articles and columns representing years. There is no single flat table to load directly.

# This is what the raw Excel looks like before convertdata.R runs
sheet_names <- excel_sheets(file.path("data", "journal IF", "JEL_Data.xlsx"))
cat("Number of sheets:", length(sheet_names))
cat("First 5 sheets:", head(sheet_names, 5))

# Each sheet looks like this:
# rows    = individual articles published in that journal
# columns = years (how many citations that article received each year)
# bottom  = summary stats: mean, SD, SE of the mean

convertdata.R loops through every sheet, extracts the mean and standard error from the bottom rows, and assembles a single clean data frame.

Data Wrangling — Filtering to the KMS Top 30

The full dataset has 232 journals. The paper focuses on 30 specific journals pre-selected by Kalaitzidakis, Mamuneas & Stengos (2003) — an earlier study that identified leading outlets in economics based on citation data from 1994–1998. We filter to exactly those 30 by name.

jIF <- journalIF %>%
  filter(
    !is.na(IF),              # remove journals with no impact factor data
    IF > 0,                  # remove journals with zero citations
    journal %in% top30names  # keep only the KMS 30
  ) %>%
  arrange(desc(IF)) %>%      # sort highest IF first
  mutate(rank = row_number()) # assign point ranks 1 through 30

The arrange and mutate steps give us the point estimate of each journal’s rank — the rank you would assign if you ignored uncertainty entirely.

The Key Statistical Step — csranks

This is the core of the replication. The csranks() function takes the point estimates and their uncertainty, runs a bootstrap simulation, and returns confidence sets for ranks.

# Convert SE vector to a diagonal covariance matrix
# (csranks requires a matrix, not a vector)
# diag(SE^2) places variances on the diagonal, zeros off-diagonal
# This assumes estimation errors are independent across journals
Sigma <- diag(jIF$SE^2)

CS <- csranks::csranks(
  jIF$IF,          # point estimates (impact factors)
  Sigma = Sigma,   # covariance matrix of estimation errors
  coverage = 0.95, # 95% confidence level
  stepdown = TRUE, # stepdown procedure (more powerful than single-step)
  simul    = FALSE, # marginal (not simultaneous) confidence sets
  R        = 1000, # number of bootstrap draws
  seed     = 101   # for reproducibility
)

# Output: CS$L = lower bound of rank confidence set
#         CS$U = upper bound of rank confidence set
jIF$CSL <- CS$L
jIF$CSU <- CS$U

What stepdown and simul Mean

Two parameters worth understanding:

stepdown = TRUE — uses a stepwise testing procedure. Instead of testing all ranks simultaneously at once, it iterates, removing journals whose ranks are already determined. This gives tighter (more informative) confidence sets than a single-step approach.

simul = FALSE — produces marginal confidence sets. Each journal’s confidence set covers its true rank with 95% probability individually. When simul = TRUE, the sets are simultaneous — all true ranks are covered jointly with 95% probability. Simultaneous sets are wider. The paper shows both; Figure 1 uses marginal sets.
